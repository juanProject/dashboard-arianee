# Arianee
The Arianee project is an independent, participative - organization whose mission is to build a global standard for the
digital certification of valuable objects by promoting and supporting the adoption of the Arianee protocol.

# Arianee dashboard
This dashboard shows several data from Arianee smart contract using the official SDK ArianeeJS.
It's a simple page running javascript to get data. 

# Charts
Arianee dashboard display 2 graphs :

- Bar chart : number of certificate since smart contract deployment divided in six columns
- Pie chart : number of transfer and token access made since smart contract deployment

# Data
Arianee dashboard display some stats about entities and transactions :

- a list of certified brands that uses Arianee 
- number of certificate created 
- token price
